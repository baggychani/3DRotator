# 3D Rotator

이미지를 3D 회전시킨 뒤 입체 그림자와 말풍선 꼬리를 더해 투명 PNG로 저장하는 로컬 웹앱입니다.

## 실행

Windows 사용자는 `start_app.bat`을 더블클릭하면 됩니다. 가상환경이 없으면 자동으로 만들고,
브라우저도 자동으로 열립니다. Python 3.11 이상이 필요하며, 실행 스크립트는 `py` 또는
`python` 명령을 자동으로 찾아 사용합니다.

`8000`번 포트가 이미 사용 중이면 다음 빈 포트를 찾아 엽니다. 프리셋은 브라우저 저장소에
묶이므로, 가능하면 매번 같은 포트로 여는 것이 좋습니다.

개발자가 직접 실행할 때는 아래 명령을 사용합니다.

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements-dev.txt
python server.py
```

## 검사

```powershell
ruff check .
```

## 구조

- `server.py`: 로컬 서버, 포트 충돌 회피, 브라우저 자동 열기
- `web/app.js`: 앱 초기화와 이벤트 연결
- `web/modules/`: WebGL 렌더링, 좌표 계산, 이미지 로딩, 컨트롤 상태, 프리셋 관리
- `web/styles/`: 기본 스타일, 레이아웃, 컴포넌트, 컨트롤, 반응형 분리

## 현재 기능

- 3D 회전, 알파 실루엣 그림자, 말풍선 꼬리 기본값과 섹션별 리셋
- 사용자 프리셋 저장, 선택, 삭제
- 잘림 방지 자동 맞춤
- 밝은 배경, 체크보드, 어두운 배경 미리보기 테마
- 말풍선 꼬리 핸들 드래그 조작
- 사용자 프리셋 파일 내보내기/불러오기
- 투명 PNG 다운로드 (`원본파일명_3D.png`)

## 렌더링 메모

- 출력 캔버스는 `2160x3840`으로, 인스타그램 스토리 규격의 2배 해상도를 사용합니다.
- WebGL2를 사용할 수 있으면 mipmap과 anisotropic filtering을 적용하고, 불가능하면 WebGL1 + linear filtering으로 동작합니다.
- 텍스처 업로드 전 이미지 외곽에 1px 투명 패딩을 추가해 회전된 가장자리의 계단/점선 아티팩트를 줄입니다.

## 프리셋 저장

사용자 프리셋은 브라우저의 `localStorage`에 저장됩니다. 같은 브라우저에서 같은 주소
(`http://127.0.0.1:8000` 등)로 다시 열면 서버를 껐다 켜도 유지됩니다.

브라우저 저장소 정책상 포트가 달라지면 다른 사이트로 취급됩니다. 예를 들어 `8000`에서
저장한 프리셋은 `8001`로 열었을 때 보이지 않을 수 있으니, 가능하면 같은 포트로 실행하세요.

## 모바일 사용

데스크톱에서는 현재 3열 레이아웃과 `2160x3840` 출력 캔버스를 유지합니다. 화면 폭이 좁은
Android 모바일에서는 CSS가 단일 컬럼으로 전환되고, 미리보기는 9:16 비율을 유지하며,
버튼과 슬라이더는 터치하기 쉽게 커집니다. 손가락 입력에서는 말풍선 꼬리 핸들의 터치 판정도
더 넓게 잡습니다.

같은 Wi-Fi 안에서 휴대폰으로 임시 테스트하려면 PC에서 아래처럼 실행한 뒤, 휴대폰 브라우저에서
PC의 내부 IP와 포트로 접속합니다.

```powershell
python server.py --host 0.0.0.0
```

예: `http://192.168.0.12:8000`

공개 링크로 모바일에서도 쓰려면 GitHub Pages 같은 정적 호스팅에 배포하면 됩니다. 이 앱은
대부분의 기능이 브라우저 안에서 동작하므로, `web/` 폴더를 정적 사이트로 배포하는 방식이
가장 단순합니다.

## 프리셋 동기화

프리셋은 기본적으로 브라우저의 `localStorage`에 저장되어 기기마다 따로 관리됩니다. PC와
Android 휴대폰 사이에서 같은 설정을 쓰려면 프리셋 영역의 `프리셋 내보내기`로 파일을 저장한 뒤,
다른 기기에서 `프리셋 불러오기`로 가져오면 됩니다.

## GitHub Pages 준비

루트의 `index.html`은 GitHub Pages에서 저장소 루트로 접속했을 때 `web/` 앱으로 이동시키는
진입 파일입니다. `.nojekyll`도 포함되어 있어 Pages가 정적 파일을 그대로 서빙합니다.
